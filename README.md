# TadpoleDBHub_APIServer
- Tadpole DB Hub API Server는 Tadpole DB Hub( https://github.com/hangum/TadpoleForDBTools )의 서브 프로젝트 입니다. 
- 올챙이(Tadpole DB Hub)에서 쿼리를 저장하고, 쿼리의 결과를 API 서버로 호출하여 받아 올수 있습니다.
